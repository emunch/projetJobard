Liens
Cette page liste les documents utilisés pour le développement d'OpenGL Mathematics.

OpenGL.org
Le site officiel d'OpenGL. On y retrouve les spécifications d'OpenGL, des news et beaucoup d'autres informations.

OpenGL Registry
La page qui contient toutes les spécifications des extensions d'OpenGL.

OpenGL 2.0 Specification
La spécification d'OpenGL 2.0.

OpenGL Shading Language Specification
La spécification d'OpenGL Shading Language

How to create extensions ?
Un document qui explique comment créer une extension OpenGL.

Extension template
Un support commenté de spécification d'extensions.

Extension example
Un exemple de référence de spécification d'extensions.

OpenGL syntax rules
Un document qui explique la syntaxe de l'API OpenGL. Ce document contient toute les abbréviations.

GLSL Extension conventions
Un document qui pose les conventions pour l'écriture d'extensions de GLSL. Ces règles forment la base du système d'extensions de GLM.
